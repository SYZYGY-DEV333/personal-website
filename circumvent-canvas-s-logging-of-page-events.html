Circumvent Canvas's Logging of Page Events
Created
Jan 24, 2021 9:15 PM
Tags
Coding
School
(Reposted from an old blog of mine)
Canvas by Instructure is a common classroom management program, used by countless schools and teachers around the world. Overall, it's a well-designed program that strikes a good balance between intuitiveness and flexibility. However, it's not without its drawbacks.
Canvas has a feature which logs browser activity when taking a quiz. The intentions are benign enough: they want to be able to track if a student leaves the browser tab or window, which would potentially imply that the student had been using outside resources inappropriately. However, there are two major downsides that I'm aware of.
Firstly, I get a little worried about the privacy implications of a website tracking how you're using your browser.
Secondly, and possibly more importantly, the Canvas logging system also detects if you were inactive for more than 30 seconds while taking the quiz, and sends the same message to the teacher as if you left the page. The implications are obvious: if doing something that requires longer than 30 seconds of thought, the teacher might suspect you were cheating.
Clearly, that poses several issues, and I wanted to try experimenting with ways of bypassing that clearly flawed system. Essentially, I wrote a script that, first of all,
1.
Detects if you're on Canvas (easy enough)
2.
Detects if you're taking a quiz (surprisingly challenging).
This is where things get tricky: I wanted to block the detection of whether the webpage was "in focus" (aka currently being used). Unfortunately, all the usual ways of fooling a website into thinking it's in focus didn't work, probably since Canvas implemented their own way of checking. Another idea: Canvas sends messages to their servers from your browser about activity/inactivity. I tried intercepting and blocking these messages, but ultimately decided it wasn't worth the effort. This left me with one option: try to screw up the timers they have running in the browser to detect 30 seconds of inactivity, 15 seconds away from the tab, etc. This proved to be relatively straightforward, as all I had to do was get rid of their ways of measuring time. This did, however, have one amusing side-effect: the quiz seems to think it's being taken during the Vietnam War (New Years Eve, 1969). However, once submitted to the teacher, the dates correct themselves, and there's nothing to indicate anything out of the ordinary from their perspective.
Installation Instructions
If you would like to install the script, which I helpfully named "What? I didn't click away from Canvas!" then follow these steps:
1.
Install a UserScript manager. On Safari or Chrome, I recommend Tampermonkey. On Firefox, I recommend Greasemonkey, though it's possible to install Tampermonkey there too.
2.
Use THIS LINK to install the UserScript and follow the appropriate prompts.
3.
Navigate to Canvas, and click on the Tampermonkey/Greasemonkey icon in the browser toolbar to make sure that the "What? I didn't click away from Canvas!" Script is running. If so, you're good to go!

window.scriptActive = false;
// Watch for changes to the webpage
const observer = new MutationObserver(mutations => {
mutations.forEach(mutation => {
mutation.addedNodes.forEach(node => {
// Check if we're actually taking a quiz.
if (node.id == 'quiz_urls') {
// The way Canvas detects inactivity or window/tab switching is through timers. If we screw up
// their way of detecting time, it ruins those timers.
// There IS the side-effect that the website seems to think you're taking the quiz during the Vietnam war, but eh, whatever.
// That nonsense doesn't make it into what the teacher sees.
Object.defineProperty(Date.prototype, 'getTime', { value: () => 45, configurable: false, writable: false });
Object.defineProperty(performance, 'now', { value: () => 45, configurable: false, writable: false });
Object.defineProperty(Date, 'now', { value: () => 45, configurable: false, writable: false });
window.scriptActive = true;
observer.disconnect();
console.log('Activated anti tab switch detector');
}
});
});
});
observer.observe(document.documentElement, {
childList: true,
subtree: true
});
